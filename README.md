# Rhinogobius Species Catalogue

Static front end for the current Rhinogobius valid-species workbook.

## Publish with GitHub Pages

In the repository settings, open **Pages**, select **Deploy from a branch**, then choose the publishing branch and the `/docs` folder.

The exported JSON intentionally does not expose local PDF file paths. Entries with a locally held PDF are marked as `本地文献`; CAS reference entries link to the public CAS record.
